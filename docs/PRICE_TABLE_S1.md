## 나머지 5종목 주간 종가 + 일간 변동 패턴 요약

### 광산 조합 (자원/광업 — 수혜)
- 패턴: 마법석과 유사하나 폭은 60% 수준. 감산 주체라서 수혜.
- W1-2: 완만 상승 (+3~10%), W3-4: 본격 상승, W5-6: 고점, W7-8: 소폭 하락 안정

### 무기 대장간 (방산 — 긴장 수혜)
- 패턴: 완만 지속 상승. 급등은 없으나 꾸준히 우상향.
- W5 외교 결렬 시 +5% 점프. 이후 안정.

### 신목 농업 (농업 — 안정)
- 패턴: 거의 횡보하다가 미세 상승. 마법석 비의존 → 안전자산 역할.
- 최대 일간 변동 ±1%.

### 연금술 공방 (화학 — 혼조 → 하락)
- 패턴: W1-2 소폭 상승 (원자재 가공 수요), W3부터 원가 상승 압박으로 하락 전환.
- W5-6에서 급락 (-5~8% 일간), W7-8 바닥 안정.

### 마도 통신 (IT/통신 — 비용 타격)
- 패턴: 마법석 가격과 역상관. 마법석 기반 통신망 운영비 폭등.
- W1-2: 미미한 하락, W3-5: 본격 하락, W6: 저점, W7-8: 소폭 반등.

## 일간 가격 생성 공식 (프로그래밍용)

```
기본 변동 = 랜덤(-0.5%, +0.5%) — 일상 노이즈
트렌드 변동 = 주차별 방향 가중치
이벤트 변동 = 뉴스 발행일에 +3~10% 추가 (종목별 상이)

최종 가격 = 전일가 × (1 + 기본변동 + 트렌드변동 + 이벤트변동)
```

MVP에서는 이 테이블을 **하드코딩**하고, 뉴스 발행 시간에 맞춰 해당 일자 가격을 반영.
