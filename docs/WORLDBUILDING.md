# 마도거래소 세계관 설계서

> 최종 수정: 2026-06-14

---

## 1. 대전제

- 유저는 **판타지 세계만 본다**. 현실 매핑은 시즌 종료 후 공개
- 캐릭터는 **정보의 매개체**이지 투자 판단의 주체가 아님
- 유저가 스토리를 읽으며 단서를 **스스로 추론**하는 것이 핵심 쾌감
- 두 가지 재미의 동시 충족: **소설 읽는 재미** + **주식 하는 재미**

---

## 2. 세계: 아르카 대륙

마법이 일상인 세계. 모든 마법의 연료는 "마법석(마나 결정체)".
마법석 없이는 통신도, 운송도, 에너지도 작동하지 않는다.

### 2.1 종족 국가

| 종족 | 영토 | 현실 매핑 (비공개) | 핵심 산업 | 세계관 내 포지션 |
|---|---|---|---|---|
| **드래곤 제국** | 북부 대륙 | 미국 | 마법석 정제, 마도 통신 | 최대 소비국. 기축통화(골드 마르크) 발행. 군사력·기술력 1위 |
| **용린 왕국** | 동부 해안 | 일본/한국 | 마수 운송, 연금술 공방 | 기술력 높으나 자원 전량 수입. 마법석 의존도 극심 |
| **엘프 연합** | 서부 삼림 | 유럽(EU) | 정령 에너지, 신목 농업 | 자체 에너지원 보유. 동맹 구조 복잡. 독자 노선 |
| **드워프 자치령** | 중앙 산맥 지하 | 중동(OPEC) | 광산 조합, 무기 대장간 | 마법석 80% 산출. 과점. 가격 결정권 보유 |
| **오크 부족연합** | 남부 황무지 | 소련/러시아 | 군사, 자원 채굴 | 자원 풍부하나 기술 낙후. 드워프와 경쟁 |

### 2.2 산업 (= 종목/테마주)

| 판타지 산업 | 현실 매핑 (비공개) | 주요 소속 | 특성 |
|---|---|---|---|
| 마법석 정제 | 정유/에너지 | 드워프(생산), 드래곤(정제) | 대륙 핵심 자원. 가격이 모든 것을 좌우 |
| 마도 통신 | IT/통신 | 드래곤 제국 | 마법석 기반 통신망. 마법석 가격에 비용 연동 |
| 마수 운송 | 자동차/운송 | 용린 왕국 | 마수 사육 + 마법석 연료. 직격탄 취약 |
| 정령 에너지 | 대체에너지/원자력 | 엘프 연합 | 마법석 비의존. 위기 시 수혜 |
| 연금술 공방 | 화학/소재 | 용린 왕국 | 원자재 가공. 원가 상승 vs 수요 증가 혼조 |
| 광산 조합 | 자원/광업 | 드워프 자치령 | 감산 시 수혜. 가격 결정 카르텔 |
| 무기 대장간 | 방산/중공업 | 드워프 자치령 | 긴장 고조 시 수혜 |
| 신목 농업 | 농업/식품 | 엘프 연합 | 비교적 안정적. 마법석 비의존 |

## 3. 캐릭터 시스템

### 3.1 핵심 원칙

- 캐릭터는 **자기 볼 일**(의뢰, 거래, 연구)을 하며 대륙을 돌아다님
- 여행 중 **우연히 목격하거나 전해 듣는** 장면이 투자 단서가 됨
- 캐릭터가 "이건 사야 해" 같은 투자 발언을 하는 것은 **금지**
- 대신 각자의 **직업적 관점**에서 자연스러운 반응만 보여줌

### 3.2 주인공 일행

| 캐릭터 | 종족 | 직업 | 여행 이유 | 자연스럽게 전달하는 정보 |
|---|---|---|---|---|
| **카이** | 드래곤(인간형) | 마법석 감정사 견습생 | 스승의 의뢰 수행 (각지 광맥 조사) | 광맥 상태, 채굴량, 마법석 품질 변화 |
| **도르** | 드워프 | 은퇴 광부 출신, 카이의 보디가드 | 고향 근처라 길 안내 겸 동행 | 과거 vs 현재 비교, 산업 현장 분위기, 광부들 정서 |
| **리나** | 용린 | 소규모 상단 소속 상인 | 거래처 순회 (물건 사고팔며 이동) | 물가 변동, 운송비 변화, 도시마다 다른 시장 분위기 |
| **셀레나** | 엘프 | 정령술사 겸 학자 | 고대 정령 유적 조사 | 에너지 흐름 이상, 자연 변화, 장기적 징조 |
| **그롬** | 오크 | 용병, 정보꾼 | 일감 찾으며 따라옴 | 뒷골목 소문, 군사 동향, 검증 안 된 루머 |

### 3.3 정보 전달 방식 (= 떡밥 메커니즘)

| 유형 | 설명 | 예시 |
|---|---|---|
| **목격** | 캐릭터가 직접 본 현상 | 카이: "3번 갱도가 폐쇄되어 있었다" |
| **엿듣기** | NPC 대화를 우연히 들음 | 광부들: "어제 3번 갱도 폐쇄됐잖아" |
| **체감** | 상인으로서 가격 변동 체감 | 리나: "운송비가 저번 달보다 2배" |
| **비교** | 과거 경험과 대조 | 도르: "내가 일하던 시절엔 이런 적 없었는데" |
| **학술적 관측** | 장기 데이터 기반 분석 | 셀레나: "이 지역 마나 밀도가 3년 전부터 줄고 있어" |
| **소문** | 출처 불명, 맞을 수도 아닐 수도 | 그롬: "남쪽 술집에서 드워프 장로회가 뭔가 결정했다더라" |

### 3.4 캐릭터 간 관계 (시즌 간 연속)

- 카이-도르: 스승-제자 같은 신뢰. 도르가 과묵하게 지켜줌
- 카이-리나: 동갑 친구 느낌. 티격태격하지만 서로 의지
- 리나-도르: 세대 차이. 리나가 장난치고 도르가 한심해함
- 셀레나: 일행과 약간 거리. 학자적 냉정함. 가끔 결정적 한마디
- 그롬: 아웃사이더. 신뢰도 불확실. 근데 가끔 정확히 맞힘

## 4. 시즌 1: 마나 고갈 사태

### 4.1 현실 모티브 (비공개)
1970년대 오일쇼크. OPEC 감산 → 석유 가격 폭등 → 세계 경제 충격.

### 4.2 세계관 내 서사

드워프 자치령이 마법석 감산을 선언하고, 최종적으로 수출을 중단.
대륙 전체가 마법석 부족에 빠지며 산업 구조가 요동친다.

### 4.3 여행 루트

카이가 스승의 의뢰로 "대륙 각지 광맥의 마법석 품질 조사"를 떠남.
→ 드래곤 제국 출발 → 드워프 자치령 → 용린 왕국 → 엘프 연합 → 복귀

### 4.4 주차별 시나리오

**W1-2 (도입) — 드래곤 제국 → 드워프 자치령 도착**

- 카이 일행 출발. 평화로운 여행.
- 드워프 자치령 도착. 광산 마을 분위기가 묘하게 긴장됨
- 단서: 갱도 일부 폐쇄, 광부 수 감소, 도르의 옛 동료가 "요즘 이상하다" 발언
- 시장: 아직 큰 변동 없음. 미세한 불안 정도

**W3-4 (전개) — 드워프 자치령 체류**

- 대장로 회의 개최. 비공개. 마을 전체가 긴장
- 리나: "상단 물량 계약이 갑자기 보류됐어"
- 그롬: "장로회에서 큰 결정 났다는 소문"
- 공식 발표: "채굴 효율 점검을 위한 일시적 감산"
- 단서: "일시적"이라지만 광부들 표정이 어두움. 도르: "이건 일시적이 아니야"
- 시장: 마법석 정제↑, 광산 조합↑ 시작

**W5-6 (위기) — 용린 왕국 이동**

- 용린 왕국 도착. 마수 운송 길드가 비상회의 중
- 리나: "마법석 가격 3배 됐어. 운송 마수 먹이(마법석 사료) 값도 폭등"
- 거리에 마수 운송 마차가 줄어듦. 시민 불안
- 드워프 → 드래곤 외교 결렬 뉴스. "수출 전면 중단"
- 단서: 정령 에너지 연구소에 사람이 몰린다 (셀레나 목격)
- 시장: 마수 운송↓↓, 마법석↑↑, 정령 에너지↑ 시작

**W7-8 (결말) — 엘프 연합 → 복귀**

- 엘프 연합이 정령 에너지 수출 협상 시작
- 드워프 일부 타협 (소량 재수출), 그러나 이전 가격으로 돌아가진 않음
- 새로운 에너지 질서 형성 조짐
- 카이 스승의 의뢰 목적 밝혀짐 (떡밥 회수)
- 시장: 변동 안정화, 정령 에너지 확정 수혜, 마수 운송 부분 회복

### 4.5 시즌 1 종목별 가격 곡선 (개략)

```
         W1-2    W3-4    W5-6    W7-8
마법석    100 →   130 →   300 →   250  (폭등 후 소폭 하락)
광산조합  100 →   120 →   200 →   180  (수혜)
정령에너지 100 →  105 →   150 →   200  (후반 급등)
마수운송  100 →    95 →    50 →    65  (직격탄)
마도통신  100 →    98 →    75 →    80  (비용 상승 타격)
연금술공방 100 →  110 →   120 →   110  (혼조)
무기대장간 100 →  105 →   140 →   130  (긴장 수혜)
신목농업  100 →   100 →   105 →   110  (안정)
```

## 5. 뉴스 작성 원칙

### 5.1 여행일지 (= 스토리)

- 3인칭 관찰자 or 카이 1인칭 일기체
- 캐릭터의 행동/대화를 통해 **간접적으로** 정보 전달
- "투자하세요" 절대 안 함. "~라고 했다", "~를 봤다"로 끝남
- 감정, 분위기, 디테일로 몰입감 부여
- 한 편이 짧은 단편소설 한 장면 느낌 (500~800자)

### 5.2 마도통신 속보 (= 공식 뉴스)

- 객관적 톤. 팩트만 전달
- 여행일지와 정보가 미묘하게 다를 수 있음 (공식 vs 현장)
- "드워프 자치령 대변인: 일시적 감산" ↔ 여행일지: "광부들 표정이 어두웠다"
- → 누구 말이 맞지? = 유저의 분석 포인트

### 5.3 떡밥 설계 원칙

- **1주 떡밥**: 당일~다음 뉴스에서 회수. 바로 매매에 영향
- **2주 떡밥**: 한 페이즈 뒤에 회수. 기억하는 유저에게 유리
- **시즌 떡밥**: 마지막에 회수. 전체 스토리 완성감
- 떡밥을 심을 때 → 유저가 "뭔가 이상한데?"라고 느끼도록 **약간의 위화감** 삽입
- 회수할 때 → "아! 그때 그거!" 하는 **연결의 쾌감**

## 6. 유저 경험 흐름

```
뉴스 도착
  → 여행일지 읽기 (소설 재미 + 단서 탐색)
  → 속보 읽기 (공식 정보 + 여행일지와 대조)
  → "이 종목 오를까 내릴까?" 추론
  → 예측 기록 + 매매 실행
  → 다음 뉴스에서 결과 확인
  → "맞았다!" or "놓쳤네, 저 복선이 그거였구나"
  → 다음 뉴스가 기다려짐 (스토리 + 투자 양쪽)
```

**두 가지 재미의 결합:**
- 스토리만 읽어도 재밌음 (캐릭터, 세계관, 미스터리)
- 투자만 해도 재밌음 (예측, 적중, 랭킹)
- 둘을 결합하면 → **스토리를 잘 읽은 사람이 투자도 잘한다**

## 7. 시즌 종료 후 공개 요소

- "이번 시즌의 모티브: 1970년대 오일쇼크"
- 현실 사건 ↔ 게임 사건 매핑표 공개
- "마법석 = 석유, 드워프 자치령 = OPEC" 등
- 유저 반응: "아 그래서 그때 그랬구나!" → 금융 리터러시 자연 습득
- 분석 리포트에 현실 대응 주석 추가
